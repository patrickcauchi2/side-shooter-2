FAQ

Can I use this with React / Next.js?

Yes. Install Tailwind via npm and add the classes to your JSX. The utilities work the same.

Is the CDN version okay for production?

It’s fine for demos. For production, set up Tailwind’s build so unused classes are removed (tree-shaken) for performance.

How do I add dark mode?

This template uses darkMode: 'class'. Toggling the dark class on <html> switches themes.

How do I split this into pages?

Duplicate this file to about.html, pricing.html, etc. Keep the same header/footer and update links.
